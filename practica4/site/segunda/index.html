WEB 1.0. Características

  1. Sitios direccionales y no colaborativos.
  2. Pocos productores de contenidos.
  3. Muchos lectores de estos contenidos.
  4. La actualización de los sitios no se realiza en forma periódica.
  5. Interacción mínima reducida a formularios de contacto, inscripción boletines, etc.
  6. Formularios HTML enviados vía correo electrónico. Un usuario llenaba un formulario y después de hacer clic se enviaba a través de un cliente de correo electrónico, con el problema que en el código se podía observar los detalles del envío del correo electrónico.
  7. Páginas estáticas.
